# 🤖 Emoji AI System

An interactive emoji simulation where each emoji has its own personality, memory, and behavior.

## 🚀 Live Demo
👉 https://anshikamisraxx.github.io/emoji/

---

## ✨ Features

- 👁️ Eye tracking (follows cursor)
- 🧠 Memory system (emotions change over time)
- 🎭 Unique personalities per emoji
- 🔗 Chain reaction (emojis influence each other)
- ⏳ Cooldown system (emotions reset over time)
